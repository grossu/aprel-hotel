# Сайт арт-отеля «Апрель»

Статический адаптивный лендинг на основе визуального кода Non-Objective.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер из этой папки:

```sh
python3 -m http.server 8000
```

После запуска сайт будет доступен по адресу `http://localhost:8000`.

## Состав

- `index.html` — структура и контент;
- `styles.css` — дизайн-система и адаптивные стили;
- `script.js` — меню, лист ожидания и анимации;
- `assets/` — визуализации и фирменные материалы проекта.

## Демонстрационный лист ожидания

Форма собирает email и выбранный интерес к проекту только для проверки сценария. Тестовые записи сохраняются локально в браузере посетителя и никуда не отправляются. Перед реальным сбором контактов потребуется подключить отдельный сервис и опубликовать политику обработки персональных данных.
